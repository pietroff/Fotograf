Portfolio Designer Lite plugin allows you to create, manage, edit and design portfolio and showcase with few clicks.


== Description ==

Portfolio Designer Lite is a good handy and free solution for everyone who is looking to showcase photography collections, freelance work, travel gallery, etc. Portfolio Designer Lite provides variety of 2 different portfolio layouts to setup your portfolio page for any WordPress websites. It's not easy to change or modify portfolio layout very easily, only you can do it by modifying code and css files.

However, using Portfolio Designer plugin you can design your portfolio page as per your choice to give it a *WOW* factor. Also you can modify various settings very quickly from admin side as Portfolio Designer plugin has User Friendly Admin Panel. So, beginners can start showcasing work within 10 minutes, no coding skill required!!

You can show your new portfolio page design with any page via below shortcode.

**Shortcode** : [wp_portfolio_designer_lite] - To display portfolio on page

== Installation ==

1. Upload the **portfolio-designer-lite.zip** file Via WordPress Admin > Plugins > Add New,
2. Alternately, upload **portfolio-designer-lite** folder to the /wp-content/plugins/ directory via FTP,
3. Activate the **Portfolio Designer Lite** plugin from Admin > Plugins.


== Frequently asked questions ==
= Does Portfolio Designer Lite give me total control of my portfolio page? =
Yes, Portfolio Designer gives you total control of your portfolio page. You need to just setup everything correctly and play with it. You may get more than your expectations!

= Does Portfolio designer provide some default portfolio layouts? =
Yes, Portfolio designer provides 2 default portfolio layouts - Grid and Masonry.

And in [Portfolio Designer](https://www.solwininfotech.com/product/wordpress-plugins/portfolio-designer/) PRO version, there are 3 different portfolio layouts available with 50+ combinations to design your portfolio page with unique way. In Pro version, you can design your single portfolio post pages too.

= Can I apply this portfolio designer only for my hot categories? =
Yes, there is one backend option available as you can select multiple portolio categories and portfolio page will show that categories only on frontend.

= Can I manage my "Show All" text on filtration? =
Yes, you can change that text and decorate it also whatever you like.

= What about custom post type support? =
Portfolio Designer Lite plugin provides support of 'Custom Post Type' too.

And in [Portfolio Designer](https://www.solwininfotech.com/product/wordpress-plugins/portfolio-designer/) PRO version, you can create new custom post type as well to start your new portfolio section with different post-types.

= Where can I get support or talk for portfolio designer issues? =
If you get stuck, you can ask for help with Portfolio Designer plugin [Support Forum](http://wordpress.org/support/plugin/portfolio-designer-lite) or can create a ticket at our [Support Portal](http://support.solwininfotech.com).

= Will Portfolio Designer work with my themes? =
Yes, Portfolio Designer will work with any WordPress themes, but may require some styling changes to make it compatible. If you are facing any compatibility issue with Portfolio Designer plugin then please contact us at [Support Forum](http://wordpress.org/support/plugin/portfolio-designer-lite).
